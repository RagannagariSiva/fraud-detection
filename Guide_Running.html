One-Time Setup

1Go to project folder
cd ~/Documents/fraud-detection
cd %USERPROFILE%\Documents\fraud-detection
2Create virtual environment
python3 -m venv venv
python -m venv venv
3Activate and install dependencies
source venv/bin/activate
pip install -r requirements.txt
venv\Scripts\activate
pip install -r requirements.txt
4Place the dataset (creditcard.csv from Kaggle)
mkdir -p data/raw
mv ~/Downloads/creditcard.csv data/raw/creditcard.csv
mkdir data\raw
move %USERPROFILE%\Downloads\creditcard.csv data\raw\creditcard.csv
Verify: data/raw/creditcard.csv must be a file, not a folder.

Reset Dashboard to Zero

cd ~/Documents/fraud-detection
rm -f data/fraud_stats.db
echo -n "" > logs/fraud_alerts.jsonl
echo -n "" > logs/transactions.jsonl
cd %USERPROFILE%\Documents\fraud-detection
del data\fraud_stats.db
type nul > logs\fraud_alerts.jsonl
type nul > logs\transactions.jsonl
Then restart Streamlit — dashboard resets to 0.
